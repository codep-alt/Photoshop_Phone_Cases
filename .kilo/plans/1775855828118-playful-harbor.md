# Generate Images Feature Plan

## Overview
Generate product mockup images from Lightspeed CSV orders by placing designs onto phone case mockup PSDs (front/back/side), changing case colors, and exporting as PNGs.

---

## 1. Data Flow

```
CSV (Lightspeed) → Parse Order → Find Mockup (model + variant) → Find Design Image 
                 → Open PSD → Place Design → Change Case Color → Export PNG
```

### CSV Parsing (per row)
- **Model**: Extract from `NL_Title_Long` - everything before "case" → "Samsung Galaxy A23"
- **Color**: Extract from `NL_Title_Long` - between model and "case" → "beige"
- **Design**: Extract from `NL_Title_Long` - after " - " → "Marmer waves"
- **Variant**: Extract from `NL_Variant` - before "_" → "Solid"
- **Brand**: Use `Brand` column → "Casimoda"

---

## 2. Folder Structures (Read-Only Reference)

### Mock-ups (read)
```
Mock-ups/{PhoneModel}/{VariantType}/{PhoneModel}/
├── {ColorFolder}/ (e.g., "Solid Beige", "Solid Black", "Solid Brown")
│   ├── Front.psd
│   ├── Back.psd
│   └── Side.psd
```

### Designs (read)
```
Designs/{Brand}/ (e.g., "Casimoda", "Casevibes", "Mooie Telefoonhoesjes")
├── {DesignName}.jpg/png
```

### Output (write)
```
{UserSelectedOutputFolder}/{BatchName}/
├── {OrderID}_Front.png
├── {OrderID}_Back.png
├── {OrderID}_Side.png
```

## 5. Key Algorithms

### Title Parsing
```
Input: "Samsung Galaxy A23 beige case - Marmer waves"
1. Split by " - " → ["Samsung Galaxy A23 beige case", "Marmer waves"]
2. From part[0], extract before " case" → "Samsung Galaxy A23 beige"
3. Split part[0] by " case" → ["Samsung Galaxy A23", "beige"]
   - Model: "Samsung Galaxy A23"
   - Color: "beige"
4. Design: part[1] → "Marmer waves"
```

### Color to Folder Mapping
```
Input: color="beige", variant="Solid"
1. Look up colorMappings["beige"] → "#F5F5DC"
2. Build folder: "Mock-ups/Samsung Galaxy A23/Solid/Samsung Galaxy A23/Solid Beige"
   - Variant + " " + capitalized(color) → "Solid Beige"
```

### Design File Matching
```
Input: designName="Marmer waves", brand="Casimoda"
1. Normalize: "marmerwaves" (lowercase, no spaces/special chars)
2. List files in Designs/Casimoda/
3. Match by normalized name without extension
4. Return full path if found
```

## 6. Error Handling

- **Missing mockup**: Skip order, log warning with orderId
- **Missing design**: Skip order, log warning with orderId
- **Invalid CSV row**: Skip, log warning
- **PSD processing error**: Catch and log, continue with next order
